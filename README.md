# [Ebuild Alins Template](https://github.com/alinsjs/ebuild-template-alins)

[Alins](https://github.com/alinsjs/alins) template

### 1. create-alins

```
npm create alins
```

### 2. clone

```
git clone https://github.com/alinsjs/ebuild-template-alins.git
```

### 3. ebuild-cli

```
npm i ebuild-cli -g
```

```
ebuild init <name>
```


## Install And Dev

```
cd <alins-project>
```
